[//]: # (title: How-to-log-in)

To log in into a system:

1. Install Infra CLI
```bash
brew install infrahq/tap/infra
```

2. Login to Infra

```bash
infra login SERVER
```
{prompt="$"}

![Alt](infra.png)

3. See what you can access

Run an infra list to view what you have access to:

```bash
infra list
```
